# QA Fast Learning Bot v2.1 — Умный помощник QA-инженера 🧪

Telegram-бот для подготовки к сертификации ISTQB и развития навыков тестирования.

## Возможности

| Раздел | Команда | Описание |
|--------|---------|----------|
| 🎯 **Квиз ISTQB** | `/quiz` | Вопросы из официальных ISTQB-программ, 3 уровня сложности |
| 📚 **Теория** | `/theory` | 7 тем по официальным материалам ISTQB |
| 🛠️ **Инструменты** | `/tools` | Каталог инструментов тестировщика |
| 📝 **Тест-кейсы** | `/testcase` | AI генерирует тест-кейсы |
| 🐛 **Баг-репорт** | `/bugreport` | AI помогает оформить баг |
| 💼 **Собеседование** | `/interview` | Вопросы для подготовки к интервью |
| 📋 **Чеклист** | `/checklist` | Генерация чеклиста тестирования (AI) |
| ❓ **Вопрос** | `/ask` | Любой вопрос по тестированию |

## Источники материалов

- **ISTQB CTFL Syllabus v4.0.1** — Foundation Level
- **ISTQB CTFL Mobile Application Tester Syllabus**

## Быстрый старт

```bash
pip install -r requirements.txt
```

Создайте `.env`:
```
QA_BOT_TOKEN=ваш_токен
OPENROUTER_API_KEY=ваш_ключ
```

```bash
python -m qa_bot.bot
```

## Деплой

```bash
bash deploy.sh
```
